# gemini-chat-bot-example

Simple gemini chatbot wrapper.

## Running Backend

### Backend Prerequisite's

* Install the [GO Programming Language](https://go.dev/dl/)

```bash
go install github.com/gin-gonic/gin@latest
```

```bash
go run src/api/main.go
```

## Running Frontend

### Frontend Prerequisites

* Install the [Node.js JavaScript Runtime Environment](https://nodejs.org/en/download)
* Install the [pnpm package manager](https://pnpm.io/installation)

```bash
pnpm run dev -- --open
```
